# Tutorial: Receive Acceleration Data with TouchDesigner

You can receive the OSC/JSON data with any apps supporting TCP or UDP protocol. In this chapter, we'll introduce how to receive the acceleration data using TouchDesigner.

# 1. Prerequisite

  • Install TouchDesigner
  • Get the IP address of the PC

# 2. Run ZIG SIM

  1. Launch ZIG SIM app.
  2. Go to Settings tab.
  3. Set IP Address and Port Number.
  4. Activate Acceleration tab.
