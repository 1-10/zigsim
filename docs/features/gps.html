# GPS

GPS command detects GPS signal and outputs latitude and longitude.

# Outputs

  • OSC Address:/(deviceUUID)/gps
    Values are in the order of latitude and longitude.
  • JSON key:
    • latitude:gps.latitude
    • longitude:gps.longitude
